XG7Lobby 1.1

XG7Lobby é um plugin feito por DaviXG7. Nele há vários recursos essenciais de um hub para Minecraft! Funciona em versões acima da 1.8


Dependências: https://www.spigotmc.org/resources/protocollib.1997/

Repositório: https://github.com/DaviXG7/XG7Lobby/tree/master/src/main

Suporte: https://ko-fi.com/davixg7

Inventários personalizados:

Recursos:

Essencial ao lobby:

  • Scoreboard;
  • Tablist;
  • Bossbar;
  • Posicionamento do lobby;
  • Customização de menu;
  • LauchPad;
  • Itens com ações personalizadas;
  • Foto e MOTD;
  • Eventos de entrada personalizados;
  • Anúncios.

Ações:

  • Ações:
  • Título (TITLE);
  • Subtítulo (SUBTITLE);
  • Título e subtítulo (TITSUBTIT);
  • Abrir inventário (OPEN);
  • Anunciar (BROADCAST);
  • Invocar (SUMMON);
  • Efeito (EFFECT);
  • Executar um comando no jogador (COMMAND);
  • Executar um comando no console (CONSOLE);
  • Teleporte (TP);
  • Fazer o jogador voar (FLY);
  • Manda uma mensagem ao jogador (MESSAGE);
  • Toca um som (SOUND);
  • Manda um ActionBar (ACTIONBAR);
  • Troca um item de um slot para outro (SWAP);
  • Esconde os jogadores (HIDE);
  • Mostra os jogadores (SHOW).
  • TAGS:
  • PLAYER -> Retorna ao nome do jogador;
  • PERMISSION -> Executa a ação só se o jogador estiver com a permissão.

Moderação:

  • Mute;
  • TempMute;
  • Dados dos jogadores (Silenciado, infrações, etc);
  • Ban, tempban;
  • Warn;
  • Ações por permissão.

Chat:

  • Anti-spam;
  • Bloqueador de palavrões;
  • Bloqueador de comandos.
  • Ban, tempban;
